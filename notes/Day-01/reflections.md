What is a database? - It is an organized collection of digital data stored in computer systems so that it can be easily accessed, updated and maintained. 

Why not use Excel for everything? - Excel has a limited functionality where the user cannot access it via other applications and it lacks the knowledge of maintaining the data when multiple user accesses it. 

What problems does a DBMS solve? - It helps to organize data in efficient way, relates the data using keys, can be intelligent to have a decision when there is a duplicate/irrelevant entry, and easily integrated with the applications. 

How does this relate to the Oracle databases you work with? - All these come from the day to day experience of working with databases. 

The Primary key in the students database is the student_id. If it has the same value in any other rows, the DBMS does not allow the user to insert the contents into a new row.
